# Oncosimis Website

This is the official website for Oncosimis Biotech, a company pioneering novel biologic therapies.

## Website Stack

- **HTML5**: Structure and content
- **CSS3**: Styling with custom properties (CSS variables) for theming
- **JavaScript**: Basic interactivity and animations
- **Assets**:
  - Images: Logo and favicon in `assets/images/`
  - Videos: Hero background video in `assets/videos/`
- **Third-party integrations**:
  - Google Fonts (Montserrat and Lato)
  - Font Awesome for icons
  - Google Maps embed for location

## Features

- Responsive design with mobile-first approach
- Fixed header with navigation
- Hero section with video background
- Sections for mission, science, pipeline, products, news, awards
- Embedded Google Map in footer
- Social media links
